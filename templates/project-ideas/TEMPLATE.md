# 新项目 Idea 生成模板

> 按此模板填写，生成的新项目 prompt 放入 `templates/project-ideas/` 目录，
> 按序号+slug命名（如 `01-slug.md`），并更新 [`00-ROADMAP.md`](./00-ROADMAP.md) 的学习路径。

---

## 项目编号与名称

按 ROADMAP 中已有项目续号，格式 `序号-slug`。

## 所属分层

<!-- 三选一 -->

- [ ] Layer 1: 基础设施 / 工具链
- [ ] Layer 2: 模型评估 / Prompt 工程
- [ ] Layer 3: 模型推理 / 部署 / 检索

## 难度评估

<!-- 选一个，并附一句话理由 -->

- [ ] ⭐ 简单（单文件 CLI 工具，< 4 个 task）
- [ ] ⭐⭐ 中等（多模块，< 8 个 task）
- [ ] ⭐⭐⭐ 困难（完整系统，≥ 8 个 task，可能涉及多语言/TUI/GPU）

## 语言 / 技术栈

<!-- 选一种或多种，标推荐程度 -->

- [ ] Python（推荐：AI 生态最成熟，新手友好）
- [ ] Rust（推荐：性能关键路径 / TUI / 系统级）
- [ ] TypeScript（推荐：Web 生态 / MCP 协议 / Node 工具链）
- [ ] Go（推荐：高并发网络服务）
- [ ] Shell（仅实验/脚本项目）
